C语言的操作符

发表于 2015-12-05   |   分类于 C   |     |   阅读次数

按位操作符

在使用右移位操作符>>时，要注意当对signed类型的带符号值进行右移时，某些机器将对左边空出的部分用符号位填补（即“算术移位”），而另一些机器则对左边空出的部分用0填补（即“逻辑移位”）。

掩码操作

一元操作符~用于求整数的二进制反码。例如：

x = x & ~077

将把x的最后六位设置为0。注意，表达式x&~077与机器字长无关，它比形如x&0177700的表达式要好，因为后者假定了x是16位的数值。这种可移植的形式并没有增加额外开销，因为~077是常量表达式，可以在编译时求值。

下面是一个函数getbits(x,p,n)，它返回x中从右边数第p位开始向右数n位的字段。假定最右边的一位是第0位。getbits(x,4,3)返回x中第4、3、2三位的值。

unsigned getbits(unsigned x, int p, int n)
{
    return (x >> (p + 1 - n)) & ~(~0 << n);
}

统计一个整形数值为1的二进制位的个数

表达式x&=(x-1)可以删除x中最右边值为1的一个二进制位。

int bitcount(unsigned x)
{
    int b;
    for(b = 0; x != 0; b++)
        x &= (x - 1);
    return b;
}

赋值操作符

大多数二元操作符都有一个相应的赋值操作符op=，如果expr1和expr2是表达式，那么

expr1 op= expr2

等价于

expr1 = (expr1) op (expr2)

它们的区别在于，前一种形式expr1只计算一次。注意，在第二种形式中，expr2两边的圆括号是不可缺少的。例如，

x *= y + 1

的含义是

x = x * (y + 1)

而不是

x = x * y + 1

条件表达式

采用条件表达式可以编写出很简洁的代码。例如，下面的这个循环语句打印一个数组的n个元素，每行打印10个元素，每列之间用一个空格隔开，每行用一个换行符结束（包括最后一行）：

for(i = 0; i < n; i++)
    printf("%6d%c", a[i], (i%10==9 || i==n-1) ? '\n' : ' ");
